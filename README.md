# GitHub Project Creator Script

_read the source, luke_

1. Create a GitHub token with the correct permissions (admin:org, admin:org_hook, project, repo)
2. Create a template project and find the project's number in the URL
3. Copy `.env.example` to `.env` and configure environment variables
4. Update `inputfile.txt` with a list of `eID,GitHub Username` pairs (e.g. "russfeld,russfeld-student")
5. Run the script using `run.sh` command
